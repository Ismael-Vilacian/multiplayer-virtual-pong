<h1>Online Multiplayer Pong</h1>
</br>
This project involves a simple game inspired by the well-known Pong, with the primary goal of exploring client/server communication technologies. The idea behind this game is to implement, using socket technologies, the capability for two players to play and compete simultaneously online. The emphasis is on studying online interaction between users, utilizing socket infrastructure to deliver an interactive and collaborative gaming experience. </br>
